# Apple AI Signals 与 Agent 环境

> 参考: https://research.frankk.site/apple-ai-signals

## 核心观点

Agent 能力的真正决定因素在**环境**。

环境的评价标准主要是两个维度：
1. **速度** - 快的环境可以让 Agent 在短时间内进行大量实验
2. **数据** - 好的数据可以提供推理所需的上下文

## iOS 系统作为 Agent 环境

iOS 系统（目前看起来）就是最好的一个 Agent 运行环境：
- 提供了丰富的用户交互场景
- 拥有完善的数据收集机制
- 具备快速的反馈循环

这解释了为什么 Apple 在 AI 领域的布局值得关注。
